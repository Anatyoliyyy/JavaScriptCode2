Игра известная как 'Шарики' или 'Цветные линии'
Папка Data содержит:
      Records.txt - файл с рекордами
      Saves       - здесь хранятся сохранённые игры
      Balls       - картинки для шариков (можно заменять на другие)
      Menu        - картинки для меню (можно заменять на другие)
code.zip          - исходники на VB6. Папка Data должна находиться там же где и
                    извлечённые исходные коды.
Программа работает с коммандной строкой:
lines2k n         - n - количество различных шариков (1 - ...)
                    Внимание, в папке Data/Balls тогда должно быть n файлов формата
                    1.jpg,...,n.jpg

Ошибки ещё имеются, особенно при обрабоке рекордов. Пожалуйста сообщайте о них на мой
e-mail.
